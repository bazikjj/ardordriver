# Ardordrivers
Turorial for installation Windows on ardor/celvo laptops

Needed for installation:
1. usb flash
2. pc or laptop with internet access
3. driver for disk
4. BRAIN

Also, i made a video tutorial on YouTube ([link](https://www.youtube.com/watch?v=YU56cV45sg8))

1. Select you laptop on site [link](https://ardor-gaming.com/category/34770/noutbuki/)
2. Douwnload file with name IRST
3. Unpuck it in usb disk with windows unpacked iso
4. Select IRST driver in disk manager
